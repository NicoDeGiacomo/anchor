# Changelog

All notable changes to Anchor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-02-10

### Added

- Five built-in grounding modes: Panic, Anxiety, Sadness, Anger, and Grounding
- Three-phase phrase structure (Preparation, Confrontation, Reinforcement) based on Self-Instructional Training
- Custom modes: create your own modes with personalized names
- Custom phrases: add, edit, or hide phrases for any mode
- Multi-language support: English, Spanish (Español), and Portuguese (Português)
- Five themes: Auto (system), Black, Dark, Light, and White
- Cross-platform: iOS, Android, and Web
- Privacy-first: all data stored locally, no accounts, no analytics, no tracking
- Fully offline: no internet connection required
- Open source under MIT license
- Evidence-based grounding techniques reviewed by mental health professionals
